GIT Commands

--> $git status To check what needs to commit  
--> $ ls To see list of files in local directory.  
--> $git add (file name) Adding single file to staging area before commeting.  
--> $git add . Adding multiple files to staging area before commeting.  
--> $git commit -m 'your message' Commiting or adding message for future reference what we changed in the files.  
--> $git push Uploading files to live repository.
